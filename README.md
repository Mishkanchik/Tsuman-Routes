# Tsuman Bus Schedule

Beautiful animated website showing bus schedules for Tsuman village.

## Features

- Modern design with gradient background
- Animated elements using Framer Motion
- Glass-morphism card effects
- Responsive layout
- Clear separation between routes from/to Tsuman
- Dynamic card sizing based on content

## Routes

### From Tsuman
- Tsuman → Lutsk
- Tsuman → Kiversti
- Lypne → Rivne (Tsuman is a transit point)

### To Tsuman
- Lutsk → Tsuman
- Rivne → Lypne

## Technologies

- React 18
- Vite
- TailwindCSS
- Framer Motion
- Lucide React
